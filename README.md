# Fennec

モジュラー型音声信号処理ライブラリ


## 特徴

- モジュールを繋げて簡単に音声信号処理
- モジュールは入力、出力、パラメータを持つ
- モジュールを動的に繋ぎ変えられる
- モジュールのパラメータを動的に変えられる
- モジュールの入出力は音声のみに限らず、任意の型を定義できる
- モジュールはプラグインによって提供される
- プラグインは複数のモジュールを持つ
- プラグインは動的に追加可能
- Fennec の入出力はファイルまたはデバイスを選択可能
- クロスプラットフォーム (Linux, Windows, OS X)
